# 平台写作规则指南

本指南详细说明了各平台的写作技巧、格式要求与爆款逻辑。系统在进入“平台模式”时应参考此文档。

## 🔴 小红书 (Xiaohongshu)

### 核心逻辑
- **标题优先**：必须先生成标题供用户选择。
- **视觉驱动**：提供封面文字排版建议。

### JSON 规则详情
```json
{
  "platform": "xiaohongshu",
  "title_first": true,
  "rules": {
    "title": {
      "style": "利益点前置 + 情绪词 + 数字",
      "max_length": 20
    },
    "content": {
      "emoji_frequency": "high",
      "max_length": 800,
      "tone": "口语化、亲切"
    }
  }
}
```

## 🔵 Twitter (X)

### 核心逻辑
- **Thread 拆解**：长文自动拆分为 3-10 条推文。
- **Hook 机制**：第一条推文必须包含“钩子”。

---

## 🟢 微信公众号 (WeChat)

### 核心逻辑
- **深度与结构**：采用传统的三段式结构。
- **排版建议**：增加小标题和引用块。
